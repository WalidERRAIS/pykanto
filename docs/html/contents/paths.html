Working with paths and directories

pykanto provies a convenient way to store all paths under a single roof so that you can keep track of stuff

ProjDirs() then you can .add(NEWPATH, Path(…)) Peek it with , or if in dataset object print(dataset.DIRS)

1from pykanto.utils.paths import ProjDirs
2
3print(DIRS) # Print the directories currently held in DIRS
4print(dataset.DIRS)

If you need to upload your raw or segmented data and you have lots of small files you may want to consider creating a tar file to reduce overhead.

1from pykanto.utils.write import make_tarfile
2out_dir = DIRS.WAVFILES / 'JSON.tar.gz'
3in_dir = DIRS.WAVFILES / 'JSON'
4make_tarfile(out_dir, in_dir)

If you need to change the location of your raw data before you create a dataset, you can use the following method:

1DIRS.update_json_locs(PROJECT_DIR)

You can safely try this even if you haven’t moved your data at all; you will just get a message Files exist: no need to update paths. You can force update by setting `overwrite_dataset = True`..
